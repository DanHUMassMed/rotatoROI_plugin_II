RotatoROI_II Plugin Setup Instructions

The RotatoROI_II Plugin processes .nd2 files (Nikon NISâ€‘Dimensional 2 format) and relies on a specific directory structure to locate these image files.

Before Processing (Process Preparation)

  • A [root] directory for your process must be created.
    • This is where the RotatoROI_II Plugin will search for files and create output.
    • This directory can be anywhere on your hard drive.
    • Use a naming convention that aligns with the overall experiment.
      • For example: [root] = /Users/dan/MitoPaper_imageprocessing/2024-04-19

The First Pipeline Step (Process .nd2)

INPUT

  • The 'Process .nd2' step expects to find .nd2 files in the [root]/raw/[experimental condition]/ directory described above.
  • Within [root]/raw, you must create additional [experimental condition] subdirectories to hold the .nd2 files.
  • Each .nd2 file is expected to contain 3 channels:
    • Channel 1: DIC
    • Channel 2: 488
    • Channel 3: 561

OUTPUT

  • The output of the 'Process .nd2' step is a new directory: [root]/channel_split
  • This directory will contain subdirectories for each channel:
    • [root]/channel_split/488/[experimental condition]
    • [root]/channel_split/561/[experimental condition]
    • [root]/channel_split/DIC/[experimental condition]
  • Note: Channel not found will be ignored.
